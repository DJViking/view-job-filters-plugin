Es kann nützlich sein diesen Filter hinter allen anderen Filtern zu platzieren, vor allem wenn diese Filter alle oder keine Jobs selektieren könnten. Abhängig von seiner Konfiguration kann dieser Filter dann eine der folgenden Aktionen durchführen:
  • Alle Jobs hinzufügen, wenn keine Jobs ausgewählt sind.
  • Alle Jobs entfernen, wenn alle Jobs ausgewählt sind.
In allen anderen Fällen hat dieser Filter keine Auswirkung. Das heißt dieser Filter fungiert als Rückversicherung, dass die aktuelle Ansicht entweder niemals leer ist oder niemals alle Jobs enthält.
