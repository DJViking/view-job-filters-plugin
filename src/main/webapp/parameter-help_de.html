Dieser Filter erlaubt es parameterisierte Jobs anhand ihrer Parameter zu filtern. Reguläre Ausdrücke können dabei auf die Namen, Werte und Beschreibung der Parameter angewandt werden. Alternativ können auch die Vorgabewerte verglichen werden.

Wenn Name, Wert oder Beschreibung leer gelassen wird, heißt das, dass dieses Feld irrelevant ist - genauso als ob als der Reguläre Ausdruck .* verwendet worden wäre.

Dieser Filter funktioniert für folgende Parameter-Typen: Text, Boolscher Wert, Datei, Auswahl and validierender Text.
